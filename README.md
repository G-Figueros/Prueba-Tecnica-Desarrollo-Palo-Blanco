# Prueba-Tecnica-Desarrollo-Palo-Blanco
Prueba técnica para plaza de desarrollador fullstack donde se están aplicando tecnologías como Angular JS, Node JS y MySQL

1. Descargar el repositorio e instalar dependencias del node modules (npm i)
2. Abrir una terminal dentro de la carpeta frontend-angular
3. ejecutar el comando ng serve -o
4. Abrir una nueva terminal sin cerrar la anterior y entrar en la carpeta backend-node
5. ejecutar el nomando node index.js
6. Ejecutar el archivo SCRIPT_BD dentro de WorkBench MySQL
